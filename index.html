Criador de Gráfico KYRALY

Instruções de uso:

  1. Selecione um arquivo CSV utilizando o botão "Escolher arquivo". Certifique-se de que o arquivo esteja no formato CSV.
  2. Escolha o tipo de gráfico que deseja gerar a partir do seu arquivo CSV.
  3. Clique no botão "Enviar" para carregar os dados do arquivo CSV e gerar o gráfico.
  4. Após o gráfico ser gerado, você pode clicar no botão "Salvar como imagem ou PDF" para salvar o gráfico em seu dispositivo.
